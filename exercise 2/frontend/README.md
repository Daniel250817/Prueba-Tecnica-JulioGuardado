# Frontend React + TypeScript - Juego de Adivinanza de Razas de Perros

## Requisitos

- Node.js 18+ (recomendado: 20.x LTS o superior)
- npm o yarn

## Instalación

```bash
npm install
```

## Ejecución

### Desarrollo
```bash
npm run dev
```

El servidor de desarrollo estará disponible en `http://localhost:3000` (configurado en `vite.config.ts`).

### Build de producción
```bash
npm run build
```

### Preview del build
```bash
npm run preview
```

## Estructura

- `src/` - Código fuente
  - `components/` - Componentes React
  - `hooks/` - Custom hooks
  - `services/` - Servicios API
- `vite.config.ts` - Configuración de Vite
- `tsconfig.json` - Configuración de TypeScript
